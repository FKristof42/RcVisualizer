# RC Graph exploration by a single agent Algorithm Simulation and Visualisation

A Java application to simulate and visualise the RC Graph exploration by a single agent algorithm.  
The simulation generates a JSON representation of the graph, and the JavaFX application visualizes it.

---

## Prerequisites

Before running the project, ensure you have the following installed:

- **Java 17** (JDK 17)
- **Maven**

Make sure your `JAVA_HOME` environment variable is set to your Java 17 installation and Maven is available in your `PATH`.

---

## Usage

### 1. Simulation

Run the simulation to generate a JSON file:

```bash
java Rc.java
```

### 2. Visualization
To start the program, run:
```bash
mvn javafx:run
```
Then load one of the generated JSON files
